# Contributing to Universal Stream Archiver Suite

First off, thank you for considering contributing! 

## How Can I Contribute?

### Reporting Bugs
- Use the **Bug Report** template to submit an issue.
- Describe the steps to reproduce the bug and include your environment details (Python version, GPU model).

### Suggesting Enhancements
- If you have an idea for a new feature (e.g., supporting a new site's API), feel free to open an issue or a Pull Request.

### Pull Request Process
1. Fork the repo and create your branch from `main`.
2. Ensure your code follows the existing style (with English comments).
3. Update the `README.md` if your changes introduce new customization requirements.
4. Issue a Pull Request with a clear description of the changes.
